# Calculator in React

## App running in [Vercel](https://calculator-react-dusky.vercel.app/)

### Installation and Setup Instructions

- Clone this repository. 
- Check in your machine, `node` and `npm` installed globally. 
- Installation:
    `npm install` or `npm i`
- Run test 
    `npm test`
- Run in Local Server
    `npm start`
